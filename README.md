# Car-Sales-Dashboard
Overview
This project focuses on analyzing car sales and recall data to uncover patterns in profitability, sales performance, and system recalls.
The goal is to transform raw sales data into meaningful insights and visual dashboards that support decision-making for automotive business management.

The analysis explores:
  Which car models sell best and generate the most profit.
  The number of recalls per model and affected system.
  Customer sentiment trends across sales data.
  How monthly profit and quantity sold change over time.

Project Objectives
  Perform a clear and structured analysis of car sales data.
  Identify performance patterns by model, dealer, and month.
  Visualize recall counts by affected systems.
  Combine insights into an interactive and presentation-ready dashboard.

Tools and Technologies: IBM Cognos Analytics; Microsoft Excel
